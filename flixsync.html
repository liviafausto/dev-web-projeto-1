FlixSync

O FlixSync é uma API de registro e atualização de filmes e séries. Seus usuários podem utilizá-la para organizar seus filmes e séries em listas personalizadas, marcar quais já foram assistidos e descobrir novos conteúdos por meio de recomendações personalizadas.

O projeto está sendo desenvolvido em Java com o Spring Framework e utiliza um banco de dados PostgreSQL configurado com Dockerfile. Para facilitar a manutenção futura do sistema, logs estão sendo inseridos em partes estratégicas de cada serviço da API.
