Jogo de Xadrez

Esse é um dos meus primeiros projetos. Consiste em um jogo de xadrez feito inteiramente em Java, que possui todas as funcionalidades e regras do jogo implementadas - inclusive as jogadas de Roque, 'En Passant' e promoção do peão.

Foi desenvolvido a partir de um curso de Java para iniciantes, com o objetivo de praticar os conceitos base da Programação Orientada a Objetos em Java. Utiliza uma interface simples de terminal e pode ser executado em qualquer sistema operacional. Para jogar, é necessário baixar o projeto e seguir as instruções presentes no README do repositório.
